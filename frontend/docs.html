مقدمة

مرحباً بك في توثيق Relosity AI API. هذا التوثيق سيساعدك على فهم كيفية استخدام API الخاص بمنصتنا لدمج الذكاء الاصطناعي في تطبيقاتك.

Relosity AI API يوفر واجهة برمجية قوية وسهلة الاستخدام للتفاعل مع نماذج الذكاء الاصطناعي المتطورة.

المميزات الرئيسية

  • محادثة تفاعلية: تفاعل طبيعي مع AI باستخدام النصوص
  • توليد الصور: إنشاء صور فريدة باستخدام DALL-E
  • التضمينات: تحويل النصوص إلى متجهات للبحث والتحليل
  • المراجعة التلقائية: فحص المحتوى تلقائياً
  • دعم متعدد اللغات: العربية والإنجليزية

البدء السريع

اتبع هذه الخطوات البسيطة لبدء استخدام Relosity AI API:

1. إنشاء حساب

قم بإنشاء حساب جديد في منصتنا أو سجل الدخول إذا كان لديك حساب بالفعل.

2. الحصول على مفتاح API

بعد تسجيل الدخول، انتقل إلى لوحة التحكم وأنشئ مفتاح API جديد.

مثال على مفتاح API 
rel_abc123def456ghi789jkl012mno345pqr678stu901vwx234yz

3. إرسال أول طلب

استخدم مفتاح API الخاص بك لإرسال طلب إلى API:

JavaScript 
const response = await fetch('https://api.relosity-ai.com/ai/chat', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-API-Key': 'your-api-key-here'
    },
    body: JSON.stringify({
        message: 'مرحباً، كيف حالك؟'
    })
});

const data = await response.json();
console.log(data.response);

المصادقة

جميع طلبات API تتطلب مفتاح API صالح. يجب إرسال مفتاح API في رأس الطلب.

رأس المصادقة

مثال 
X-API-Key: rel_abc123def456ghi789jkl012mno345pqr678stu901vwx234yz

أخطاء المصادقة

كود الخطأ الوصف الحل
401 مفتاح API غير صالح تحقق من صحة مفتاح API
429 تم تجاوز حد الاستخدام انتظر أو قم بترقية خطتك

حدود الاستخدام

تختلف حدود الاستخدام حسب نوع الخطة:

الخطة الطلبات/دقيقة الطلبات/شهر السعر
مجاني 10 100 مجاناً
متقدم 60 1,000 $29/شهر
مؤسسي 300 غير محدود $99/شهر

عند تجاوز الحد المسموح، ستحصل على خطأ 429. انتظر حتى إعادة تعيين العداد.

محادثة AI

استخدم هذا الـ endpoint لإجراء محادثة مع الذكاء الاصطناعي.

Endpoint

POST /api/ai/chat

المعاملات

المعامل النوع مطلوب الوصف
message string نعم الرسالة المرسلة للـ AI
model string لا نموذج AI المستخدم (افتراضي: gpt-3.5-turbo)
temperature number لا درجة الإبداع (0-1، افتراضي: 0.7)
max_tokens number لا الحد الأقصى للرموز (افتراضي: 1000)

مثال على الطلب

JavaScript 
const response = await fetch('https://api.relosity-ai.com/ai/chat', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-API-Key': 'your-api-key-here'
    },
    body: JSON.stringify({
        message: 'ما هو الذكاء الاصطناعي؟',
        model: 'gpt-3.5-turbo',
        temperature: 0.7,
        max_tokens: 1000
    })
});

const data = await response.json();
console.log(data.response);

توليد الصور

استخدم هذا الـ endpoint لتوليد صور باستخدام DALL-E.

Endpoint

POST /api/ai/generate-image

المعاملات

المعامل النوع مطلوب الوصف
prompt string نعم وصف الصورة المطلوبة
size string لا حجم الصورة (1024x1024, 512x512, 256x256)
quality string لا جودة الصورة (standard, hd)
n number لا عدد الصور (1-4)

مثال على الطلب

Python 
import requests

url = "https://api.relosity-ai.com/ai/generate-image"
headers = {
    "Content-Type": "application/json",
    "X-API-Key": "your-api-key-here"
}
data = {
    "prompt": "قطة لطيفة تجلس في الحديقة",
    "size": "1024x1024",
    "quality": "hd",
    "n": 1
}

response = requests.post(url, headers=headers, json=data)
result = response.json()

print(result["images"][0]["url"])

أمثلة JavaScript

مثال شامل

JavaScript 
class RelosityAI {
    constructor(apiKey) {
        this.apiKey = apiKey;
        this.baseURL = 'https://api.relosity-ai.com';
    }
    
    async chat(message, options = {}) {
        const response = await fetch(`${this.baseURL}/ai/chat`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'X-API-Key': this.apiKey
            },
            body: JSON.stringify({
                message,
                model: options.model || 'gpt-3.5-turbo',
                temperature: options.temperature || 0.7,
                max_tokens: options.maxTokens || 1000
            })
        });
        
        if (!response.ok) {
            throw new Error(`HTTP error! status: ${response.status}`);
        }
        
        return await response.json();
    }
    
    async generateImage(prompt, options = {}) {
        const response = await fetch(`${this.baseURL}/ai/generate-image`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'X-API-Key': this.apiKey
            },
            body: JSON.stringify({
                prompt,
                size: options.size || '1024x1024',
                quality: options.quality || 'standard',
                n: options.n || 1
            })
        });
        
        if (!response.ok) {
            throw new Error(`HTTP error! status: ${response.status}`);
        }
        
        return await response.json();
    }
}

// الاستخدام
const ai = new RelosityAI('your-api-key-here');

// محادثة
ai.chat('ما هو الذكاء الاصطناعي؟')
    .then(response => console.log(response.response))
    .catch(error => console.error('Error:', error));

// توليد صورة
ai.generateImage('قطة لطيفة')
    .then(response => console.log(response.images[0].url))
    .catch(error => console.error('Error:', error));

أمثلة Python

مثال شامل

Python 
import requests
import json
from typing import Optional, Dict, Any

class RelosityAI:
    def __init__(self, api_key: str):
        self.api_key = api_key
        self.base_url = "https://api.relosity-ai.com"
        self.headers = {
            "Content-Type": "application/json",
            "X-API-Key": api_key
        }
    
    def chat(self, message: str, **kwargs) -> Dict[str, Any]:
        """إجراء محادثة مع AI"""
        url = f"{self.base_url}/ai/chat"
        data = {
            "message": message,
            "model": kwargs.get("model", "gpt-3.5-turbo"),
            "temperature": kwargs.get("temperature", 0.7),
            "max_tokens": kwargs.get("max_tokens", 1000)
        }
        
        response = requests.post(url, headers=self.headers, json=data)
        response.raise_for_status()
        return response.json()
    
    def generate_image(self, prompt: str, **kwargs) -> Dict[str, Any]:
        """توليد صورة باستخدام DALL-E"""
        url = f"{self.base_url}/ai/generate-image"
        data = {
            "prompt": prompt,
            "size": kwargs.get("size", "1024x1024"),
            "quality": kwargs.get("quality", "standard"),
            "n": kwargs.get("n", 1)
        }
        
        response = requests.post(url, headers=self.headers, json=data)
        response.raise_for_status()
        return response.json()
    
    def get_embeddings(self, input_text: str) -> Dict[str, Any]:
        """الحصول على تضمينات النص"""
        url = f"{self.base_url}/ai/embeddings"
        data = {"input": input_text}
        
        response = requests.post(url, headers=self.headers, json=data)
        response.raise_for_status()
        return response.json()

# الاستخدام
if __name__ == "__main__":
    ai = RelosityAI("your-api-key-here")
    
    # محادثة
    try:
        response = ai.chat("ما هو الذكاء الاصطناعي؟")
        print(response["response"])
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
    
    # توليد صورة
    try:
        response = ai.generate_image("قطة لطيفة تجلس في الحديقة")
        print(f"Image URL: {response['images'][0]['url']}")
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")

أمثلة PHP

مثال شامل

PHP 
apiKey = $apiKey;
        $this->baseURL = 'https://api.relosity-ai.com';
    }
    
    public function chat($message, $options = []) {
        $url = $this->baseURL . '/ai/chat';
        $data = [
            'message' => $message,
            'model' => $options['model'] ?? 'gpt-3.5-turbo',
            'temperature' => $options['temperature'] ?? 0.7,
            'max_tokens' => $options['max_tokens'] ?? 1000
        ];
        
        return $this->makeRequest($url, $data);
    }
    
    public function generateImage($prompt, $options = []) {
        $url = $this->baseURL . '/ai/generate-image';
        $data = [
            'prompt' => $prompt,
            'size' => $options['size'] ?? '1024x1024',
            'quality' => $options['quality'] ?? 'standard',
            'n' => $options['n'] ?? 1
        ];
        
        return $this->makeRequest($url, $data);
    }
    
    public function getEmbeddings($input) {
        $url = $this->baseURL . '/ai/embeddings';
        $data = ['input' => $input];
        
        return $this->makeRequest($url, $data);
    }
    
    private function makeRequest($url, $data) {
        $ch = curl_init();
        
        curl_setopt($ch, CURLOPT_URL, $url);
        curl_setopt($ch, CURLOPT_POST, true);
        curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
        curl_setopt($ch, CURLOPT_HTTPHEADER, [
            'Content-Type: application/json',
            'X-API-Key: ' . $this->apiKey
        ]);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
        curl_setopt($ch, CURLOPT_SSL_VERIFYPEER, false);
        
        $response = curl_exec($ch);
        $httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
        
        if (curl_errno($ch)) {
            throw new Exception('cURL Error: ' . curl_error($ch));
        }
        
        curl_close($ch);
        
        if ($httpCode !== 200) {
            throw new Exception('HTTP Error: ' . $httpCode);
        }
        
        return json_decode($response, true);
    }
}
